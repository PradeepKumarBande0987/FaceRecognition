# Face Recognition Final

Project scaffold for a face recognition system with robustness, fairness, security, edge, and multimodal features.

## Structure

- `main.py` - Entry point
- `api/` - FastAPI service
- `configs/` - YAML experiment configs
- `data/` - Raw/processed datasets and train/val/test splits
- `models/` - Backbones, modules, fairness, GAN, multimodal, checkpoints, exports
- `src/` - Importable source package for pipeline and utilities
- `scripts/` - Training, evaluation, export, benchmark scripts
- `tests/` - Unit, integration, performance tests
- `docs/` - Paper, API docs, figures, results
- `experiments/` - Run artifacts, ablations, hyperparameter logs
- `database/` - Local SQLite database
- `logs/` - Training, inference, access logs
